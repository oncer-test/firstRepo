# Tic-Tac-Toe Game

A simple, object-oriented Tic-Tac-Toe game implementation in Python.

## Features

- Modular, object-oriented design
- Support for AI and human players
- Comprehensive game state management
- Extensive unit testing

## Installation

1. Clone the repository
2. (Optional) Create a virtual environment
3. Install dependencies: `pip install -r requirements.txt`

## Running Tests

```bash
pytest
```

## Game Components

- `Board`: Manages game board state and win conditions
- `Player`: Handles player attributes and move generation
- `TicTacToeGame`: Coordinates game flow and player interactions
